Mis proyectos

Xopeleuak es un proyecto personal que emprendí desde la conceptualización del producto pasando por el diseño, el desarrollo del sitio web, y la comercialización. El proyecto participo en el reto Jalisco 2021 siendo seleccionado hasta la 3ra etapa del concurso y en la iniciativa posible.org el proyecto avanzo a la etapa regional. Si quieres conocer más del sitio puedes dar clic aquí.

Para hacer Xopeleuak use:

Trello, Jira y kanban para la administración del proyecto

Figma, css3, sass, bootstrap4,jade, google fonts y font awesome para el diseño del sitio web

Inkscape para la elaboración de los logos

JavaScript, html5, JQuery, atom y pug para el desarrollo web y animaciones.

Git para el control de versiones

Github-pages para el hosting del sitio

ERPnext para la gestión comercial y distribución del producto

Python, Selenium, Xpath, Testlink para el Testing

SQL para la actualización de precios y almacenamiento de logs
